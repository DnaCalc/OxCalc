## 5. Current OxFml Formatting Intake

The current OxFml W073 formatting update has been reviewed as an inbound observation only. The W039 upstream-host run was regenerated after that local update.

3. `thresholds` remains meaningful only for scalar/operator/expression rule families where threshold text is the actual input.
