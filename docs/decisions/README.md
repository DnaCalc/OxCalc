# OxCalc Decision Register

Lightweight decision tracking using `ODR-NNN` pattern (OxCalc Decision Record).

## Format

Each decision is a separate file: `ODR-NNN-<SLUG>.md`

## Template

```markdown
# ODR-NNN: <Title>

- **Status**: proposed | accepted | superseded | rejected
- **Date**: YYYY-MM-DD
- **Context**: <why this decision is needed>
- **Decision**: <what was decided>
- **Consequences**: <what follows from this decision>
- **Cross-repo impact**: <impact on OxFml/Foundation if any>
```
