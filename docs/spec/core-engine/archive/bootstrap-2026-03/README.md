# OxCalc Core-Engine Bootstrap Archive

## Purpose
This directory preserves the initial OxCalc core-engine bootstrap spec set before the
major rewrite and recollation program.

These files are retained for provenance and comparison. They are not the intended long-term
canonical shape for the rewritten OxCalc spec library.

## Archive Scope
This archive freezes the core-engine bootstrap files that were active before the new
rewrite-driven canonical set was drafted.

## Provenance
- Archive created during the OxCalc spec rewrite program.
- Source repo: OxCalc
- Archive label: `bootstrap-2026-03`

## Notes
1. Foundation snapshot files are preserved here because they were part of the bootstrap
   working set, even though they are not OxCalc-owned canonical doctrine.
2. Additional non-core-engine bootstrap docs may be archived separately if they are also
   superseded by the rewrite.
3. Rewritten canonical docs should live above this archive, not inside it.
