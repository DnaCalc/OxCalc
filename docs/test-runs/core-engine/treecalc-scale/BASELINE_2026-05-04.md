## Observations

- The graph build phase is the dominant cost once static edge volume reaches roughly 8M edges.
- Dynamic potential carriers make descriptor lowering substantially more expensive even when static edge count remains at 2M.
- Invalidation closure is cheap for the grid profiles because only two border constants are edited; it becomes visible in fanout when one anchor invalidates almost every formula node.
- `--recalc-rounds 1024` pushes synthetic calculation over 3.7s while preserving the same 1M-node, 8M-edge model and closed-form checks.
- Relative rebind churn is the current outlier: deriving rebind seeds over 1M formula owners and 8M relative descriptors took about 337.6s.
- A live process sample during the relative-rebind run showed about 11.2GB working set and 11.7GB private memory; this is an observational sample, not a formal peak-memory measurement.
